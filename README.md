# SQL for Data Analysis - Task 3

## 📌 Objective
The objective of this task is to use SQL queries to extract, manipulate, and analyze structured data from a database using MySQL Workbench.

## 🛠️ Tools Used
- MySQL Workbench
- SQL
- GitHub

## 📂 Dataset
- Sample - Superstore.csv

## 📋 Task Performed
- Imported the Superstore dataset into MySQL.
- Executed SQL queries for data analysis.
- Used filtering, sorting, grouping, and aggregate functions.
- Created a SQL view for analysis.
- Created an index to improve query performance.
- Captured query outputs as screenshots.

## 🔍 SQL Concepts Covered
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions (COUNT, SUM, AVG)
- Subqueries
- Views
- Indexes

## 📸 Output
Screenshots of the SQL queries and their outputs are included in the `screenshots` folder.
